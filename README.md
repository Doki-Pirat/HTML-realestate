About This Website — CityGate Properties
CityGate Properties is a multi-page luxury real estate website built with HTML, Tailwind CSS, and JavaScript. It consists of three pages:
Pages
🏠 Home (home.html)
The landing page features a full-screen hero image with a call-to-action button, an animated stats bar (500+ properties sold, $2B+ total sales, 30 years of experience, 98% client satisfaction), a featured listings grid showcasing 4 properties, a client testimonials section, and a footer with navigation, contact info, and social media links.
🏡 Properties (properties.html)
A dedicated listings page with filter buttons (All / Buy / Rent) that dynamically show or hide property cards. Each card displays a property image, name, location, price, and details like beds, baths, and square footage.
📬 Contact (contact.html)
A contact page with an embedded Google Maps iframe showing the office location, a contact form (first/last name, email, phone, message), and an office image alongside the form.
